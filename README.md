Multi-Class Sentiment Analysis System

Objective
This project classifies text into four categories:
- Positive
- Negative
- Neutral
- Mixed

Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK

Steps
1. Data Collection
2. Data Preprocessing
3. Feature Extraction (TF-IDF)
4. Model Training (Logistic Regression)
5. Evaluation

Features
- User input prediction
- Confusion matrix
- Model saving (joblib)
